Aula 1.2 — Conheça Windows, Linux, macOS e os S.O. móveis

🌍 Um mundo de sistemas operacionais

Existem diversos sistemas operacionais no mercado, cada um com suas características, público-alvo e propósito. Conhecer as opções disponíveis é fundamental para entender qual ferramenta se encaixa melhor em cada situação.

No segmento de computadores pessoais, três sistemas dominam o mercado: Windows, Linux e macOS. Nos dispositivos móveis, os protagonistas são Android e iOS. Nesta aula vamos conhecer cada um deles.

📊 Segundo dados de 2024, o Windows lidera com cerca de 72% do mercado de desktops, seguido do macOS (~15%) e do Linux (~4%). Nos celulares, o Android domina com aproximadamente 72% e o iOS com 27%.

🪟 Windows

🪟
Microsoft Windows
Desenvolvido pela Microsoft · Lançamento original: 1985

O Windows é o sistema operacional mais utilizado em computadores pessoais no mundo. Criado pela Microsoft, ele se tornou padrão em empresas, escolas e residências ao longo das últimas décadas, especialmente por sua compatibilidade com uma enorme variedade de hardware e software.

Principais versões ao longo da história

Windows 95 Windows XP Windows Vista Windows 7 Windows 8 Windows 10 Windows 11 ★

Características principais

  • Interface gráfica intuitiva com Menu Iniciar, barra de tarefas e Área de Trabalho
  • Ampla compatibilidade com programas, jogos e periféricos
  • Integrado com os serviços Microsoft: OneDrive, Office, Teams e Xbox
  • Suporte a Active Directory — muito usado em ambientes corporativos
  • Windows Defender embutido para proteção contra malware
✔ Pontos positivos
  • Fácil de usar
  • Suporte a mais softwares
  • Grande comunidade e suporte
  • Compatibilidade de hardware
✖ Pontos negativos
  • Licença paga
  • Mais suscetível a vírus
  • Consome mais recursos
  • Atualizações forçadas

🐧 Linux

🐧
Linux
Criado por Linus Torvalds · Lançamento: 1991 · Código aberto (open source)

O Linux é um sistema operacional gratuito e de código aberto, o que significa que qualquer pessoa pode ver, modificar e distribuir seu código-fonte. Por isso, existem diversas versões do Linux chamadas distribuições (distros), cada uma adaptada para um público ou finalidade específica.

O Linux está por toda parte: em servidores web (mais de 90% da internet roda Linux), em supercomputadores, em dispositivos Android e até no sistema da NASA e da SpaceX.

Principais distribuições (distros)

🟠
Ubuntu Ideal para iniciantes
🔴
Fedora Tecnologias recentes
🔵
Debian Estabilidade e servidores
🟢
Mint Interface familiar
⚫
Arch Avançado e personalizável
🇧🇷
Big Linux Criado no Brasil
✔ Pontos positivos
  • Gratuito e open source
  • Muito seguro e estável
  • Leve em hardware antigo
  • Alta personalização
✖ Pontos negativos
  • Curva de aprendizado
  • Menos suporte a softwares
  • Jogos com limitações
  • Drivers podem falhar

🍎 macOS

🍎
macOS (Apple)
Desenvolvido pela Apple · Exclusivo para hardware Apple · Baseado em Unix

O macOS é o sistema operacional dos computadores Apple (MacBook, iMac, Mac Mini e Mac Pro). Diferente do Windows e Linux, o macOS só funciona em hardware Apple — essa integração entre hardware e software resulta em um sistema muito estável e otimizado.

Baseado em Unix (assim como o Linux), o macOS é muito popular entre profissionais criativos: designers, editores de vídeo, músicos e desenvolvedores de software.

Características principais

  • Ecossistema Apple: integração perfeita com iPhone, iPad, Apple Watch e iCloud
  • Interface elegante, consistente e fácil de usar
  • Excelente desempenho nos chips Apple Silicon (M1, M2, M3…)
  • Muito usado em produção de música (Logic Pro), vídeo (Final Cut Pro) e design
  • Terminal Unix completo para usuários avançados
✔ Pontos positivos
  • Alta estabilidade
  • Design refinado
  • Segurança robusta
  • Integração com iPhone
✖ Pontos negativos
  • Hardware caro
  • Menos softwares
  • Não é customizável
  • Jogos limitados

📱 S.O. Móveis: Android e iOS

📱
Sistemas Operacionais Móveis
Projetados para smartphones e tablets

Os dispositivos móveis também possuem seus próprios sistemas operacionais, otimizados para telas táteis, bateria e conectividade. Os dois grandes são Android (Google) e iOS (Apple).

Característica 🟢 Android (Google) 🍏 iOS (Apple)
Código Open source (AOSP) Fechado (proprietário)
Dispositivos Samsung, Motorola, Xiaomi… Apenas iPhone e iPad
Loja de apps Google Play Store Apple App Store
Personalização Alta Média
Segurança Boa Muito alta
Atualizações Varia por fabricante 5+ anos garantidos
Preço Diversas faixas de preço Premium (mais caro)

🔍 Comparativo: Windows vs Linux vs macOS

Critério 🪟 Windows 🐧 Linux 🍎 macOS
Custo Pago (licença) Gratuito Gratuito*
Facilidade Alta Média Alta
Segurança Média Alta Alta
Jogos Excelente Limitado Limitado
Servidores Usado Dominante Raro
Design/Criação Bom Bom Excelente
Hardware necessário Qualquer PC Qualquer PC Apenas Apple

* O macOS é gratuito, mas o hardware Apple é caro.

💡 Qual escolher? Não existe resposta única. O Windows é ideal para o uso geral e jogos. O Linux é a melhor escolha para servidores, programadores e quem quer privacidade. O macOS brilha para profissionais criativos dentro do ecossistema Apple.

📋 Recapitulando

  • Os principais S.O. de desktop são Windows, Linux e macOS
  • O Windows é o mais popular, fácil de usar e compatível com a maioria dos softwares e jogos
  • O Linux é gratuito, open source, seguro e dominante em servidores — com várias distribuições disponíveis
  • O macOS é exclusivo da Apple, muito estável e preferido por profissionais criativos
  • Nos dispositivos móveis, Android (Google) e iOS (Apple) dominam o mercado
  • A escolha do S.O. depende da finalidade de uso, do orçamento e do perfil do usuário

📝 Exercício — Múltipla Escolha

Leia cada situação e escolha o Sistema Operacional mais adequado.

1. Um estudante quer jogar os últimos lançamentos de games e precisa de máxima compatibilidade com títulos comerciais. Qual S.O. é mais indicado?

AWindows — maior base de games e suporte de fabricantes de GPUs
BLinux — gratuito e leve para rodar jogos
CmacOS — melhor desempenho gráfico
DAndroid — sistema mais rápido
✔ Correto! O Windows domina o mercado de games com suporte nativo de fabricantes de hardware e estúdios de jogos.
✖ Não é o melhor para games. Embora o Linux tenha melhorado com o Proton/Steam, e o macOS tenha boa GPU, nenhum se compara ao ecossistema do Windows.

2. Uma empresa precisa de um servidor web estável, seguro e gratuito para hospedar seu site. Qual S.O. é o mais utilizado nesse cenário?

AWindows Server — mais fácil de configurar
BLinux — gratuito, estável e dominante em servidores
CmacOS Server — muito seguro
DAndroid — baseado em Linux
✔ Correto! Mais de 70% dos servidores web do mundo rodam Linux — é gratuito, estável, seguro e altamente configurável.
✖ O Windows Server existe, mas tem custo de licença. O Linux é a escolha dominante no mercado de servidores.
✖ O macOS Server foi descontinuado pela Apple. Além disso, exige hardware exclusivo da Apple.
✖ O Android é baseado no kernel Linux, mas é projetado para dispositivos móveis, não para servidores web.

3. Maria comprou um smartphone Samsung. Qual sistema operacional mobile ela provavelmente está usando?

AiOS — sistema da Apple
BWindows Mobile — sistema da Microsoft
CAndroid — sistema do Google, usado pela Samsung
DLinux puro — sistema open source
✔ Correto! A Samsung usa Android (com a interface One UI por cima). O iOS é exclusivo do iPhone/Apple.
✖ O iOS é exclusivo da Apple. Apenas iPhones e iPads o utilizam.
✖ O Windows Mobile foi descontinuado pela Microsoft em 2019.
✖ O Android é baseado no kernel Linux, mas é um sistema próprio. Linux puro não é usado em smartphones comuns.

4. João é designer gráfico e comprou um MacBook Pro. Qual S.O. ele vai encontrar pré-instalado?

AWindows 11
BUbuntu Linux
CChrome OS
DmacOS — sistema exclusivo da Apple
✔ Correto! Todos os computadores Mac vêm com macOS pré-instalado. É um sistema exclusivo do hardware Apple.
✖ O MacBook usa exclusivamente o macOS, sistema operacional próprio da Apple. Embora seja possível instalar outros sistemas, ele vem com macOS de fábrica.

5. Qual das alternativas descreve corretamente uma característica do Linux?

AÉ um sistema pago desenvolvido pela Microsoft
BÉ gratuito, open source e disponível em distribuições como Ubuntu e Fedora
CÉ exclusivo de computadores Mac
DSó pode ser usado em servidores, não em computadores pessoais
✔ Correto! O Linux é open source (código aberto), gratuito e existe em dezenas de distribuições: Ubuntu, Fedora, Debian, Arch, Mint, entre outras.
✖ O Linux não tem relação com a Microsoft. Foi criado por Linus Torvalds em 1991 e é mantido pela comunidade open source.
✖ Exclusivo da Apple é o macOS, não o Linux. O Linux roda em qualquer hardware compatível.
✖ O Linux pode ser usado normalmente como sistema desktop com interface gráfica. Ubuntu e Linux Mint são exemplos populares para uso pessoal.

🛠️ Atividade Prática — Conhecendo Outros S.O.

⏱ ~25 min 🌐 Requer internet 📍 Sem instalação
1
Qual Windows está instalado aqui?
Pressione Windows+R, digite winver e pressione Enter. Uma janela mostra a versão exata do Windows. Anote e compare com os colegas: todos têm a mesma versão?
2
Experimente o Linux no navegador
Abra o navegador e acesse distrosea.com. Selecione Ubuntu 22.04 LTS e clique em Start. Aguarde o carregamento (pode demorar alguns segundos). Você estará usando Linux de verdade, diretamente no navegador.
3
Explore a interface do Ubuntu
Dentro do Ubuntu virtual, tente localizar: o gerenciador de arquivos, o terminal e as configurações do sistema. Clique no menu de aplicativos (grade de pontos). Anote onde encontrou cada um.
4
Compare com o Windows
Volte ao Windows e abra Configurações (Windows+I). Compare os menus de configurações dos dois sistemas. Qual é mais fácil de navegar para você? Por quê?
5
Tabela comparativa
Na área de anotações abaixo (ou no papel impresso), escreva 3 semelhanças e 3 diferenças que você observou entre o Windows e o Ubuntu.
📌 Para saber: Embora a aparência seja diferente, Windows e Ubuntu realizam as mesmas funções básicas: gerenciar arquivos, executar programas e configurar o hardware. A maior diferença está no modelo de desenvolvimento: Windows é proprietário e pago; Ubuntu é open source e gratuito.
