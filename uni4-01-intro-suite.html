📦 Introdução à Suíte de Escritório

Aula 4.1 — Visão geral das suítes LibreOffice, Microsoft Office e Google Workspace: componentes, características e diferenças entre as plataformas

💻 O que é uma Suíte de Escritório?

Uma suíte de escritório é um conjunto integrado de aplicativos de produtividade projetados para trabalhar em conjunto. Em vez de programas isolados, a suíte oferece ferramentas complementares — editor de texto, planilha e editor de apresentações — com interface consistente e compatibilidade entre os arquivos.

No mercado de trabalho, no ambiente escolar e na vida cotidiana, dominar ao menos uma suíte de escritório é considerado uma competência fundamental de informática. Os documentos criados por essas ferramentas estão presentes em relatórios, provas, currículos, planilhas de orçamento e apresentações profissionais.

💡 As três grandes suítes atuais — Microsoft Office, Google Workspace e LibreOffice — cobrem as mesmas necessidades básicas, mas com modelos de negócio, armazenamento e recursos distintos.

📅 Uma breve história

1979 — WordStar
Primeiro processador de texto comercialmente bem-sucedido para computadores pessoais.
1983 — Lotus 1-2-3
A planilha que transformou o PC numa ferramenta de negócios indispensável.
1990 — Microsoft Office
Lançamento da primeira versão do Office (Word + Excel + PowerPoint), que domina o mercado até hoje.
2002 — OpenOffice.org
Primeira suíte open source de grande adoção, tornando a produtividade acessível gratuitamente.
2006 — Google Docs
Pioneiro na edição colaborativa em tempo real direto no navegador, sem instalação.
2011 — LibreOffice
Fork do OpenOffice, mantido pela The Document Foundation. Tornou-se a principal alternativa gratuita ao MS Office.

💻 Microsoft Office em Detalhes

💻
Microsoft 365 (antes Office 365)

O Microsoft Office é o padrão da indústria no mundo corporativo. Disponibilizado em versão de assinatura mensal/anual (Microsoft 365) ou licença permãnente (Office 2021/2024). A versão Microsoft 365 Education é gratuita para alunos de instituições cadastradas.

  • Plataformas: Windows, macOS, iOS, Android e navegador (office.com)
  • Formatos nativos: .docx, .xlsx, .pptx (padrão universal)
  • Armazenamento: OneDrive (1 TB incluso no 365)
➕ Vantagens
  • Padrão da indústria
  • Recursos mais avançados
  • Compatibilidade universal
  • Suporte oficial robusto
➖ Desvantagens
  • Custo (assinatura)
  • Requer ativação/licença
  • Mais pesado no hardware

📱 Google Workspace em Detalhes

📱
Google Workspace (antes G Suite)

O Google Workspace é gratuito com qualquer conta Google (@gmail.com). Funciona 100% no navegador, salvando automaticamente no Google Drive. A versão Workspace for Education é gratuita para escolas e integra-se nativamente ao Google Classroom.

  • Acesso: docs.google.com, sheets.google.com, slides.google.com
  • Destaque: colaboração em tempo real com múltiplos usuários simultaneamente
  • Armazenamento: 15 GB gratuitos no Drive (compartilhado entre Gmail e fotos)
➕ Vantagens
  • Completamente gratuito
  • Colaboração em tempo real
  • Salva automaticamente
  • Acessível de qualquer dispositivo
  • Integrado ao Classroom
➖ Desvantagens
  • Requer internet
  • Menos recursos offline
  • Dados armazenados no Google

🔓 LibreOffice em Detalhes

🔓
LibreOffice — The Document Foundation

O LibreOffice é um software livre e de código aberto (licença MPL 2.0), mantido por uma comunidade global. Funciona completamente offline, sem necessidade de conta ou assinatura. É muito usado em administrações públicas e instituições de ensino ao redor do mundo.

  • Plataformas: Windows, Linux, macOS — gratuito para sempre
  • Formato nativo: ODF (.odt, .ods, .odp) — padrão ISO aberto
  • Compatibilidade: lê e salva em .docx, .xlsx, .pptx sem problemas
➕ Vantagens
  • 100% gratuito e sem assinatura
  • Funciona offline
  • Open source e transparente
  • Preserva privacidade
➖ Desvantagens
  • Interface menos moderna
  • Compatibilidade imperfeição eventual
  • Sem colaboração nativa

📄 Formatos de Arquivo

Cada suíte tem seu formato nativo, mas todas conseguem ler e salvar nos formatos principais. Saber escolher o formato certo evita problemas de compatibilidade:

.docx
Word Document
MS Word • Google Docs • LibreOffice Writer
.xlsx
Excel Spreadsheet
MS Excel • Google Sheets • LibreOffice Calc
.pptx
PowerPoint Presentation
MS PowerPoint • Google Slides • Impress
.odt
OpenDocument Text
LibreOffice Writer • Formato aberto ISO
.ods
OpenDocument Spreadsheet
LibreOffice Calc • Formato aberto ISO
.pdf
Portable Document Format
Universal • Leitura em qualquer dispositivo

⚠️ Regra prática: Use .docx / .xlsx / .pptx para enviar arquivos que precisam ser editados por outras pessoas. Use .pdf para documentos finais (currículo, relatório, prova) — ele garante que o layout será preservado em qualquer dispositivo.

✅ Boas Práticas

💾
Salve com frequência Use Ctrl+S regularmente. No LibreOffice e Word, ative o salvamento automático. No Google, já é automático.
📋
Nomeie bem os arquivos Use nomes descritivos: "Relatorio_Mensal_Jan2025.docx" é melhor do que "documento1.docx".
📁
Organize em pastas Crie uma estrutura de pastas por projeto, disciplina ou período. Evite deixar tudo na área de trabalho.
📤
Exporte como PDF para enviar Ao enviar documentos finais por e-mail ou imprimir, sempre exporte para PDF para garantir o layout.
🔄
Mantenha backups Use o Drive ou OneDrive para sincronizar automaticamente. Para arquivos locais, copie periodicamente para pen drive.
⚖️
Use a ferramenta certa Texto longo → Writer/Word. Dados e cálculos → Calc/Excel. Apresentações visuais → Impress/PowerPoint.

🛠️ Atividade Prática — Explorando as Três Suítes

⏱ ~30 min 🌐 Requer internet 💻 PC
1
Acesse o Google Docs
Abra o navegador e acesse docs.google.com com sua conta Google (ou conta institucional do IFRN). Crie um novo documento em branco. Digite um parágrafo curto apresentando-se: nome, curso e o que espera aprender nesta disciplina.
2
Baixe e abra no LibreOffice
No Google Docs, vá em Arquivo → Fazer download → Microsoft Word (.docx). Abra o arquivo baixado no LibreOffice Writer. O texto foi preservado? A formatação foi mantida? Anote as diferenças que perceber na interface.
3
Compare as interfaces
Com os dois programas abertos lado a lado (ou alternando entre eles), identifique: Onde fica o menu de formatação de fonte? Como se salva o arquivo em cada um? Onde está a opção de inserir imagem? Anote 3 semelhanças e 3 diferenças.
4
Explore o Office Online (bônus)
Acesse office.com com uma conta Microsoft (pode criar gratuitamente). O Microsoft Word Online está disponível sem instalação. Abra o mesmo documento .docx. Compare com o Google Docs e o LibreOffice. Qual interface parece mais familiar para você?
5
Exporte como PDF
No LibreOffice Writer, vá em Arquivo → Exportar como PDF. No Google Docs, use Arquivo → Fazer download → PDF. Abra o PDF gerado e verifique se o layout está correto. Observe que o PDF não pode ser editado diretamente.
📌 Para refletir: As três suítes fazem coisas muito parecidas. A escolha entre elas depende de contexto: Google Docs é ideal para colaboração e acesso móvel; LibreOffice para trabalho offline sem custo; Microsoft Office quando o ambiente já usa o ecossistema Microsoft. Saber usar as três é um diferencial profissional.
