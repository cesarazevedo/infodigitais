📊 Planilhas: Gráficos e Colaboração

Aula 4.10 — Tipos de gráficos, criação, personalização, minigráficos, compartilhamento no OneDrive e proteção de planilhas no Microsoft Excel

📊 Tipos de Gráfico

O Excel oferece dezenas de tipos de gráfico, mas na prática a maioria dos casos é atendida por seis tipos principais. A escolha correta do tipo depende da natureza dos dados e da mensagem que você quer transmitir.

Colunas Comparar valores entre categorias (ex.: vendas por produto, notas por disciplina).
Barras Igual a colunas, mas horizontal. Melhor quando os nomes das categorias são longos.
Linhas Mostrar tendências ao longo do tempo (ex.: faturamento mensal, temperatura diária).
Pizza Mostrar proporções de um todo. Use apenas com até 5–6 fatias para manter a clareza.
Dispersão (XY) Identificar correlações entre duas variáveis numéricas (ex.: altura × peso).
Área Como o de linhas, mas com a área preenchida. Enfatiza o volume acumulado ao longo do tempo.

➕ Inserindo e Editando Gráficos

Para criar um gráfico no Excel, selecione os dados (incluindo cabeçalhos) e acesse Inserir → Gráficos Recomendados. O Excel sugere o tipo mais adequado para os dados selecionados — é um bom ponto de partida antes de explorar outros tipos.

👉 Passo a passo

  1. Selecione os dados incluindo os cabeçalhos de linha e coluna.
  2. Acesse Inserir → Gráficos Recomendados ou escolha diretamente o tipo no Ribbon.
  3. O gráfico é inserido na mesma planilha. Clique e arraste para reposicioná-lo.
  4. Use as alças nos cantos para redimensionar.
  5. Clique com o botão direito no gráfico → Mover Gráfico para enviá-lo a uma planilha dedicada.

💡 Atalho rápido: selecione os dados e pressione Alt+F1 para inserir um gráfico de colunas imediatamente na planilha atual. Pressione F11 para criar o gráfico em uma planilha separada.

🎨 Elementos e Personalização

Ao clicar em um gráfico, surgem duas abas no Ribbon: Design do Gráfico e Formato. Também aparecem três botões flutuantes ao lado direito do gráfico para gestão rápida de elementos, estilos e filtros.

Título do gráfico Clique no título para editá-lo. Pode ser vinculado a uma célula digitando =A1 na barra de fórmulas.
Eixos Eixo horizontal (categorias) e vertical (valores). Formato, escala e título de cada eixo são personalizáveis.
Legenda Identifica as séries de dados. Pode ser posicionada acima, abaixo, à direita ou à esquerda do gráfico.
Rótulos de Dados Exibe os valores diretamente sobre as barras ou pontos. Essencial para gráficos de pizza e quando o eixo não é exibido.
Linhas de Grade Linhas horizontais ou verticais que facilitam a leitura dos valores. Mantenha apenas as principais para não poluir o gráfico.
Linha de Tendência Botão direito em uma série → Adicionar Linha de Tendência. Mostra a direção geral dos dados (linear, exponencial etc.).

🎭 Estilos e cores

  • Design do Gráfico → Estilos de Gráfico: galeria de estilos pré-definidos (com e sem fundo preenchido).
  • Design do Gráfico → Alterar Cores: paletas de cores monocromáticas ou coloridas para todas as séries.
  • Formatar série individualmente: clique duas vezes em uma barra ou linha específica para abrir as opções de cor, borda e efeito daquela série.
  • Alterar tipo: Design do Gráfico → Alterar Tipo de Gráfico — testa outros tipos sem recriar o gráfico do zero.

⚠️ Atenção: ao exportar o gráfico para o Word ou PowerPoint, preferência Vincular dados → se a planilha mudar, o gráfico no outro documento é atualizado. Já Colar como figura cria uma imagem estática que não atualiza.

⚡ Minigráficos (Sparklines)

Minigráficos são gráficos em miniatura inseridos dentro de uma única célula, sem precisar de uma área separada. São perfeitos para exibir tendências ao lado dos dados em tabelas de resumo.

👉 Como inserir

  1. Selecione a célula onde o minigráfico deve aparecer (ex.: G2).
  2. Acesse Inserir → Minigráficos → Linha (ou Coluna ou Ganhos/Perdas).
  3. No campo Intervalo de dados, selecione as células com os valores do período (ex.: B2:F2).
  4. Arraste a alça de preenchimento para criar minigráficos para as linhas seguintes.

☁️ Colaboração e Compartilhamento

Assim como no Word, o Excel integrado ao OneDrive permite que múltiplas pessoas editem a mesma planilha simultaneamente. O Excel Online (versão no navegador) oferece a maioria dos recursos para colaboração em tempo real.

1
Salve no OneDrive Arquivo → Salvar como → OneDrive. A planilha passa a ter uma URL própria e fica sincronizada automaticamente.
2
Compartilhe com permissões Botão Compartilhar (canto superior direito). Insira e-mails ou gere um link. Escolha: Pode editar para colaboradores, Pode exibir para leitores.
3
Edite em tempo real Cada colaborador vê os cursores dos outros em cores distintas. Alterações aparecem automaticamente sem precisar salvar manualmente.
4
Histórico de versões Arquivo → Informações → Histórico de Versões. O Excel salva snapshots automáticos. Clique em qualquer versão para visualizá-la ou restaurá-la.

🔒 Proteger a planilha

Quando a planilha é compartilhada, é importante proteger células críticas (fórmulas, cabeçalhos) contra edição acidental.

Proteger Planilha Revisão → Proteger Planilha. Define senha e permite escolher o que os usuários podem fazer (selecionar células, formatar, inserir linhas etc.).
Bloquear células específicas Selecione as células editáveis → Ctrl+1 → Proteção → desmarque “Bloqueada”. Ative a proteção da planilha. Apenas as células desbloqueadas ficarão editáveis.
Proteger Pasta de Trabalho Revisão → Proteger Pasta de Trabalho. Impede que abas sejam adicionadas, excluídas, renomeadas ou movidas.
Criptografar com senha Arquivo → Informações → Proteger Pasta de Trabalho → Criptografar com Senha. Exige senha para abrir o arquivo.

📤 Exportação e Integração

O Excel se integra facilmente com outros programas da suíte Microsoft e com outros aplicativos. Conhecer as opções de exportação e integração economiza retrabalho.

  • Exportar como PDF: Arquivo → Exportar → Criar PDF/XPS. Configure a área de impressão antes (Layout → Área de Impressão) para incluir apenas os dados desejados.
  • Exportar como CSV: Arquivo → Salvar Como → CSV. Gera um arquivo de texto simples compatível com qualquer sistema. Atenção: o CSV não preserva formatação, fórmulas nem múltiplas planilhas.
  • Copiar tabela para o Word: selecione o intervalo no Excel → Ctrl+C → no Word, use Colar Especial → Vincular e Manter Formatação de Origem para que a tabela atualize quando a planilha mudar.
  • Copiar gráfico para o PowerPoint: clique no gráfico → Ctrl+C → no PowerPoint, escolha Usar Tema de Destino e Vincular Dados.
  • Importar dados externos: Dados → Obter Dados permite importar de arquivos CSV, bancos de dados, páginas web e outros formatos, com atualização automática.

✅ Boas Práticas de Visualização

🎯
Escolha o gráfico certo para os dados Comparação → Colunas. Tendência → Linhas. Proporção → Pizza. Correlação → Dispersão. Nunca use pizza para mais de 6 categorias.
📝
Título descritivo e unidade no eixo O título deve dizer o quê, de quem e quando. O eixo vertical deve sempre indicar a unidade (R$, %, unidades etc.).
🎨
Menos é mais nas cores Use uma cor por série, sem gradientes, sombras ou efeitos 3D. Gráficos 3D distorcem as proporções e dificultam a leitura correta.
⚡
Minigráficos para tendências rápidas Em dashboards e resumos, prefira Sparklines a gráficos completos quando o objetivo é mostrar a direção (sobe/desce), não o valor exato.
🔒
Proteja as fórmulas antes de compartilhar Antes de enviar a planilha para colaboradores, trave as células com fórmulas e cabeçalhos. Deixe desbloqueadas apenas as áreas de entrada de dados.
💾
Vincule gráficos ao exportar Ao inserir gráficos do Excel no Word ou PowerPoint, prefira Vincular em vez de Colar como figura. Assim os gráficos se atualizam automaticamente.

🛠️ Atividade Prática — Dashboard de Vendas

⏱ ~55 min 💻 Microsoft Excel + OneDrive 💾 Salvar como Dashboard_SeuNome.xlsx
1
Monte a base de dados
Crie uma planilha chamada Dados. Monte uma tabela com os cabeçalhos: Mês, Produto A, Produto B, Produto C, Total. Preencha os dados de Janeiro a Dezembro com valores fictícios (ex.: entre R$ 2.000 e R$ 8.000). Na coluna Total, use =SOMA(B2:D2) e arraste. Converta em Tabela Formatada (Ctrl+T).
2
Crie um gráfico de colunas agrupadas
Selecione as colunas Mês, Produto A, Produto B e Produto C (segure Ctrl para selecionar colunas não adjacentes). Acesse Inserir → Gráfico de Colunas Agrupadas. Dê o título Vendas por Produto — 2025. Posicione o gráfico em uma nova planilha chamada Dashboard (clique com botão direito no gráfico → Mover Gráfico).
3
Crie um gráfico de linhas para o total mensal
Volte à planilha Dados. Selecione as colunas Mês e Total. Crie um Gráfico de Linhas com Marcadores. Adicione rótulos de dados (botão + ao lado do gráfico → Rótulos de Dados). Dê o título Evolução do Total Mensal. Mova também para a planilha Dashboard.
4
Adicione minigráficos na tabela
Volte à planilha Dados. Insira uma nova coluna após Total chamada Tendência. Selecione a célula F2, acesse Inserir → Minigráficos → Linha e selecione o intervalo B2:D2 como dados. Arraste a alça de preenchimento para F3:F13. Ative o ponto de destaque nas cores (Design de Minigráfico → Marcar Pontos → Ponto Alto).
5
Proteja a planilha de dados
Na planilha Dados, selecione apenas as células de valores (B2:D13). Acesse Ctrl+1 → Proteção e desmarque “Bloqueada”. Agora acesse Revisão → Proteger Planilha. Marque apenas “Selecionar células desbloqueadas”. Tente clicar em uma célula de cabeçalho — o Excel não permitirá editar. Tente editar um valor de dados — deve funcionar normalmente.
6
Salve no OneDrive e compartilhe
Salve o arquivo no OneDrive (Arquivo → Salvar Como → OneDrive). Clique em Compartilhar e gere um link com permissão Pode exibir. Copie o link e cole no campo de tarefa no AVA da disciplina. Verifique que os dois gráficos aparecem corretamente na planilha Dashboard antes de enviar.
📌 Para refletir: Um dashboard bem construído no Excel — com dados organizados em uma planilha, gráficos em outra e proteção nas fórmulas — comunica informações complexas de forma visual e imediata. As habilidades desenvolvidas ao longo das aulas 4.8, 4.9 e 4.10 formam a base para análise de dados no ambiente profissional.
